### Variables

Variables name may contain letters (upper or lower case), digits and/or underscore.
The names never start with digit.
Python is case-sensitive.

Naming conventions:

1. mon_age
2. monAge

Variable assginment:

from, global, if, import, in, is, lambda

true, try, while, with, yield
